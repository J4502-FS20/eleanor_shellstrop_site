Style and formatting

Whoa, did you see that? That had italics in the font!

That was the em tag, which emphasizes text so that a screen reader can know that the text needs to be emphasized. It emphasizes the content.

If you just need to show the text in italics but don't need to emphasize it, you can present it in italics using the i tag.

There is also the strong tag, which emphasizes text so that a screen reader can know that the text needs to be emphasized. It emphasizes the content and appears in bold.

If you just need to show the text in bold but don't need to emphasize it, you can present it in bold using the b tag.
